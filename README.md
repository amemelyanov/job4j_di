Проект job4j_di для работы с Dependency Injection

Приложение, площадка для тестирования внедрения зависимостей

Функционал:

* Внедрение зависимостей.
* Работа с context application.
* Создание с Docker образа.

Стек технологий
- Java 17
- Maven 3.8

Инструменты:
- Checkstyle
